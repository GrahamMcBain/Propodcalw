# Hey Neighbor OpenClaw Production System

Automated podcast guest outreach powered by OpenClaw AI.

## Status: Ready for Deployment

This system automates prospect discovery and personalized email campaigns for podcast guest bookings.

## Railway Deployment

The system is configured to deploy automatically on Railway with:
- OpenClaw gateway for AI automation
- PostgreSQL for data persistence  
- Telegram bot for monitoring and control
- Gmail integration for email campaigns
